Welcome to the API!

How to use this API:

  • GET /posts/:id Get a post by ID ':id' with the ID of the post you want to retrieve.
  • POST /posts/:id Create a new post. The body should be a JSON object with 'title', 'content', and 'author' keys.
  • PUT /posts/:id Update an existing post. Replace ':id' with the ID of the post you want to update. Tho body should be a JSON object with the 'title', 'content', and 'authot' keys you want to update.
  • DELETE /posts/:id Delete a post. Replace ':id' with the ID of the post you want to delete.
